Group Cheatsheet

A guide to commands in the Terminal, git, html, css and ruby.


Terminal Commands

CommandDefinition
lsList the files in the current direcotry
ls -aList the files in the current directory including any hidden files
cd Enter a new directory
cd ..Go up a directory
mkdirMake a new directory in the current location

HTML Commands

CommandDefinition

ul Unordered List (Bullets)
ol Ordered List (Numbered)
li List Item
strongBold
em Italic
br Break (new line)

Git Commands

CommandDefinition
git cloneClones files stored on github under specific github address
git initnames a folder a git repository so you can link it to github
git commit confirms that code within the git repository is committed in git (must also use -m for naming the change you have made to code
git pushsends confirmed git committed code up to github profile
git pulltakes down code from a profile on github that you have linked with (e.g. when forking someone's giithub profile


CSS Commands

CommandDefinition
background-color:Sets the background color
color:Sets the color of the text
font-size:Sets the text size in px
font-weight:Sets properties of the text e.g. whether it should be bold or not.
font-family:Sets the type of font e.g. Arial
text-alignAligns the text either center, left or right
list-style-type:You can decide the shape of the bullet point, if you want one.
visibility: Makes it visible / invisible depneding on a curser action. .e.g a hover or a click.
padding-top:Adds space (padding) to the top of the object.
border:Sets the size of the border and color (several properties can be set)

Ruby Commands

CommandDefinition
irbEnter Interactive Ruby - here you can ask different tasks of Ruby and assign variables for functions
upcaseMAKE EVERYTHING IN CAPITALS
capitalize Make the first letter a capital
if...else ifdefine conditions that your calculations will use
odd...evenruby knows if your integer is odd or even, you can ask it 2, odd? and it will return false
put or print will put out what you have put in - print "hamster" output will be "hamster"
return will give back output modified by your function
